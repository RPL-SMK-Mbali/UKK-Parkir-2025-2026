## ðŸ“‹ Prerequisites

Sebelum memulai instalasi, pastikan sistem Anda memenuhi persyaratan berikut:

### Versi PHP
- **PHP >= 8.2** (Disarankan PHP 8.2 atau 8.3)
- Pastikan ekstensi PHP berikut aktif:
  - `bcmath`
  - `ctype`
  - `curl`
  - `dom`
  - `fileinfo`
  - `json`
  - `mbstring`
  - `openssl`
  - `pcre`
  - `pdo`
  - `tokenizer`
  - `xml`

### Database
- **MySQL >= 5.7** atau **MariaDB >= 10.3**
- Atau PostgreSQL >= 10.0
- Atau SQLite >= 3.8.8

### Tools Lainnya
- **Composer >= 2.5**
- **Node.js >= 22.x**
- **NPM >= 10.x** atau **Yarn >= 1.22**
- **Git**

### Sistem Operasi
- Linux (Ubuntu 20.04+, Debian 11+)
- macOS (12.0+)
- Windows 10/11 (WSL2 disarankan)
